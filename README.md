## ðŸ“¦ Installation

### Method 1: WordPress Dashboard

1. Download the plugin ZIP file from [Releases](https://github.com/SaintHossam/whatsapp-chat/releases)
2. Go to **Plugins** > **Add New** in your WordPress admin
3. Click **Upload Plugin** button
4. Choose the downloaded ZIP file
5. Click **Install Now**
6. Activate the plugin

### Method 2: Manual Installation (FTP)

1. Download and extract the plugin ZIP file
2. Upload `saint-hossam-whatsapp-chat` folder to `/wp-content/plugins/`
3. Go to **Plugins** in WordPress admin
4. Activate **Saint Hossam WhatsApp Chat**
